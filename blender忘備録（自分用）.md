### 選択
- A : 全選択
- L : 独立したメッシュの選択
- B : 矩形選択
- C : 円で塗りつぶし選択
- Shiftキー＋左クリック : 複数選択
- Ctrlキー＋左クリック: 最短距離を選択
- Ctrlキー+左クリック＋ドラッグ: 輪っか選択
- Alt+右クリック : 辺ループ選択
- Ctrl+Alt+右クリック : 辺リング選択
- Ctrl + ＋ : 選択範囲の拡大
- Ctrl + － : 選択範囲の縮小
- Ctrl + I : 選択範囲の反転
- Alt＋Shift＋M : 非多様体選択 メッシュに穴が開いている部分（切り口的な部分）を一括で選択出来ます。
- Shift＋Ctrl＋Alt＋F : リンクするフラットな面
- Shift＋G : 類似選択
